# rsps-architecture

This repository hosts the **RSPS-RRI** (Recursive Sovereign Project Space /
Recursive Relational Intelligence) codebase as a subdirectory.  The main
project lives in `rsps-rri/` and includes:

- Android client (Kotlin, Room, Observatory/Witness/IDS components).
- Python orchestration layer (FastAPI, governance, n8n workflows).
- Architecture documentation in `rsps-rri/docs/`.

To get started, cd into `rsps-rri` and follow the instructions in its
README.

This repo is also where we stage future modules or related systems as the
architecture grows.
